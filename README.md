# Telegram Business AI-бот (Gemini)

Бот отвечает на личные сообщения от твоего имени через Telegram Business + Gemini API.

## Что нужно перед запуском

1. **Telegram Premium** на твоём личном аккаунте (обязательно для Business-функций).
2. **Бот в Telegram** — создать через @BotFather (`/newbot`), получить токен.
3. **Gemini API ключ** — из Google AI Studio 


## Настройка Business-подключения в Telegram

1. Settings → Telegram Business → Chatbots
2. Выбери своего бота (он должен быть найден по username)
3. Дай разрешения:
   - Reply to messages — обязательно
   - Mark as read — опционально
   - Delete messages — опционально

## Установка и запуск

```bash
cd tg_business_bot
pip install -r requirements.txt --break-system-packages

# Указываешь свои ключи через переменные окружения (НЕ хранить в коде/гите):
export TELEGRAM_BOT_TOKEN="токен_от_botfather"
export GEMINI_API_KEY="твой_новый_gemini_ключ"

python bot.py
```

## Как это работает

- Бот слушает `business_message` обновления (специальный тип для Business-режима)
- Каждое сообщение от собеседника передаётся в Gemini с историей переписки (хранится в `chat_history.json`)
- Ответ модели отправляется обратно через `send_message` с указанием `business_connection_id` — выглядит так, будто ответил ты сам

## Важные нюансы

- **Бесплатность**: модель `gemini-2.0-flash` имеет щедрый бесплатный тир в Google AI Studio — для личной переписки этого более чем достаточно
- **Безопасность**: никогда не публикуй и не присылай свой API-ключ кому-либо, включай его только через переменные окружения
- **Кастомизация тона**: меняй `SYSTEM_PROMPT` в bot.py, чтобы бот отвечал в твоём стиле
- **Запуск 24/7**: для постоянной работы нужен сервер/VPS (или Raspberry Pi, или просто оставленный включённым ПК) — на твоём личном компе бот будет работать только пока скрипт запущен

## Возможные доработки

- Включать автоответчик только когда ты не в сети
- Фильтровать спам перед отправкой в Gemini
- Разные системные промпты для разных контактов
- Логирование диалогов в БД вместо JSON-файла
